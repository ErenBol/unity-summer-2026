# Unity Summer 2026

This repository contains my Unity learning projects for Summer 2026.

## Projects

### 02 - Roll-a-Ball / NavMesh Enemy Project

A beginner Unity 3D project based on the Roll-a-Ball tutorial.

Features:
- Player movement with Rigidbody
- Collectible pickup objects
- Score UI with TextMeshPro
- Win condition
- Enemy that follows the player using NavMeshAgent
- Lose condition when the enemy catches the player
- Static and dynamic obstacles
- Basic materials and wall styling
